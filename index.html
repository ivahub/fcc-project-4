Print

The print() function prints the specified message to the screen, or other standard output device. The message can be a string, or any other object, the object will be converted into a string before written to the screen.

Therefore, the following code:

print('Hello World')

outputs:

Hello World

Tuples

Tuples are immutable lists, meaning, their content cannot be changed. They visually differ from lists as well, because their values are defined within regular brackets (), instead of square brackets we use for lists.

  • heroes = ('buffy', 'xander', 'willow', 'giles', 
    • 'oz', 'tara', 'dawn')
For Loops

A loop simply enables us to repeat an action (or actions) for the entire list, without having to go through each value separately.

A simple for-loop looks like this:

  • for hero in heroes:
    •  print('I love ' + hero)
